Module globalize

Give a module the ability to load itself into the global namespace.

Usage:

    -- mymodule.lua:
local globalize = require 'globalize'

local M = {}

...

globalize(M)

return M

...

-- usesmymodule.lua:
require 'mymodule'() -- This loads all of the symbols in 'mymodule' into the global namespace.
